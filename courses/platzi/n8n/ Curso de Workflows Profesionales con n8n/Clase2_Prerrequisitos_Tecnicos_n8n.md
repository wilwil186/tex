## ¿Qué Debes Saber de API y HTTP para Conectar Servicios?
Conectaremos con servicios externos, por lo que es importante comprender la interfaz de una API y el protocolo HTTP. Ambos permiten que dos aplicaciones se comuniquen mediante peticiones que envían y reciben datos. Durante el curso, se usarán los métodos GET y POST para ese intercambio.

### ¿Qué es una API y Cómo se Conecta por HTTP?
- **API**: Interfaz entre aplicaciones que facilita la conexión entre sistemas mediante HTTP.
- **HTTP**: Protocolo de comunicación usado para solicitar y entregar datos entre cliente y servidor.
- **Conectar servicios externos**: Se harán llamadas que dependen de estos estándares.

### ¿Qué Significan los Métodos GET y POST?
- **GET**: Método de HTTP para recibir datos de un servicio.
- **POST**: Método de HTTP para enviar datos a un servicio.
- **Uso en el curso**: Se utilizarán para operaciones básicas de intercambio de información.

## ¿Por Qué JavaScript y JSON son Claves en n8n?
n8n está programado en JavaScript. Conocer sus fundamentos te permitirá manejarte con expresiones en los nodos y entender cómo se transforma la información. Los datos viajan entre nodos en estructura JSON, por lo que dominar este formato es esencial para manipular correctamente la información.

### ¿Qué Fundamentos de JavaScript Necesitas para Expresiones?
- **Sintaxis base de JavaScript**: Necesaria para leer y escribir expresiones.
- **Expresiones en nodos**: Se usan para acceder, transformar y encadenar datos.
- **Flujo de trabajo**: Las expresiones controlan cómo se combinan los resultados de cada nodo.

### ¿Cómo Viajan los Datos Entre Nodos en Estructura JSON?
- **Formato JSON**: Estructura universal para representar datos como pares clave-valor.
- **De un nodo a otro**: El resultado de un nodo se entrega al siguiente en JSON.
- **Manipulación correcta**: Entender el formato evita errores al leer, modificar o combinar información.
